# ⚡ クイックスタート（3ステップで起動！）

## 必要なもの
1. Docker Desktop（無料）
2. Gemini APIキー（無料）

## 起動方法

### 1️⃣ Docker Desktopをインストール
[Docker Desktop](https://www.docker.com/products/docker-desktop/)をダウンロード＆インストール

### 2️⃣ 起動
`起動.bat` をダブルクリック

ブラウザで http://localhost:3000 が開きます！

## 停止方法
`停止.bat` をダブルクリック
